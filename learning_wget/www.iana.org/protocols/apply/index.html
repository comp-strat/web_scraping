Protocol Registration Forms

The majority of protocol registries have no special form in order to accept registration requests. For these, use the general assignments form. However, if the registry is listed on the following table you should use the specific application form or template listed.

Please note that there are varying qualification criteria for different registries. Most of these criteria are identified on our protocols matrix.

Registry Registration Procedure
Private Enterprise Numbers New PEN assignment
Modify existing PEN assignment
Multicast Addresses IPv4 Assignments
IPv6 Assignments
Media Types Media Type Application Form
Port Numbers New Assignments
Port Number Modifications
IfTypes IfType Application
SDP Names Use SDP Names Template and submit to iana@iana.org
Media Feature Tags Use Media Feature Tag Template and submit to iana@iana.org
TRIP ITAD Numbers TRIP ITAD Application
All other protocol registries General Protocol Registry Form
